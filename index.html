About me

I'm a student of Computer Science from Instituto Tecnológico y de Estudios Superiores de Monterrey (ITESM), I like to work at game design, Android design and frontEnd development. Portfolio

CREmx Project

Solidary Project I participated in my University's social service, I worked in a team and did HTML and CSS files modifications, I mostly worked at the hamburguer menu mobile implementation and the convertion of the Informe Anual to a mobile version. By clicking the icons below it redirects to a GitHub Repo that shows only the things I did, you can also watch the implementation by clicking the other icons (you need a mobile browser or open the link with a smartphone).



Ternium Project

Team's project I worked with 4 more friends and we did a web training app, the goal of it is to create videogames that can help the worker to understand how things work at Ternium. I worked mostly at the game development and design team, but also did some Rest API so the games and the database worked to show some analytics in the web app, a really challenging project I loved to work with the team.

Cáritas App

Team's project I worked with other 4 friends and we did an Android app in association with Cáritas de Monterrey, a non-lucrative and volunteer group. I worked only at Android development and design, my most important contribution was the design and programming of the "Donaciones" window were the user is able to look up the donations he has done to this association and the exact days of the transaction, the user was able to donate by touching a button in this same window.



Multiagent/Unity Project

If I can define this project it would be like the hardest one I have ever done, I worked in a team with 3 friends and we did a Python multiagent system that simulates data from a traffic light, this system was later adapted to Unity and worked with the Python program done before. I worked mostly at the Python and Unity programming, the connection with these two with a Rest API, I also helped with the Unity design and 3D model development.
